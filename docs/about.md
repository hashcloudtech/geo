# 关于我们

## 品牌介绍

**HashCloud GEO** 是 [HashCloud](https://hashcloud.cn) 旗下专注于 **GEO（Generative Engine Optimization）生成式引擎优化** 的垂直业务品牌。

母公司 HashCloud 深耕数字化服务领域，提供软件开发、云服务、智能解决方案等一站式技术服务。GEO 业务线独立运营，聚焦企业 AI 搜索可见度优化，是 HashCloud 面向 AI 时代的企业品牌战略级布局。
